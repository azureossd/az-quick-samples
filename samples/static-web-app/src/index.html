What is Azure Static Web Apps?

Azure Static Web Apps is a service that automatically builds and deploys full-stack web apps to Azure from a code repository. This sample demonstrates a modern, responsive web application that showcases the capabilities of the platform.

Key Benefits:

  • Global Distribution: Your content is served from Azure's global CDN
  • Automatic SSL: Free SSL certificates for custom domains
  • GitHub Integration: Automated CI/CD from your repository
  • Serverless APIs: Optional Azure Functions integration
  • Authentication: Built-in authentication providers
  • Custom Routing: Client-side routing support
